# MyScriptBase

Base script menu untuk:
- Teleport
- Auto Farming
- Auto Fishing (speed bisa diatur)

Script ini dibuat sebagai template:
- Mudah diedit
- Tidak di-obfuscate
- Cocok untuk belajar & develop script hub sendiri

Struktur:
- Loader.lua → Loader utama
- Main.lua → GUI & kontrol menu
- Config.lua → Semua pengaturan
- Modules/ → Isi logic fitur
